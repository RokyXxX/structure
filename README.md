🎯 **Structure** is a developer-friendly, PowerPoint-native framework that combines:

- 🔧 `coreAPI` — cores, low-level programming, APIs
- 🎨 `paperAPI` — design systems, themes, and UI blocks
- 📁 `APIs` — APIs that make everything easy

Combined, it creates a simple operating system, but complex at the very end.
